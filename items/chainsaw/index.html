Chainsaws are an Item in Escape the Backrooms. It appears to simply be a Chainsaw.

Appearance

Chainsaws looks like a typical Chainsaw with an orange handle. It cannot be used as a Weapon, despite being a Chainsaw, but it can be used to break boards.

Location

This Item can be found in:

  • The Manilla Room
  • Level 0.11
  • Level !~! - Hotel Chase

Use

Using the Item

Upon using, the Chainsaw will start moving for a short amount of time.

Purpose

If the Chainsaw comes into contact with certain wood if near it when in use, the wood will break. Some wood needs multiple uses of the Chainsaw to break.

Trivia

  • Farmer Facelings can be seen holding Chainsaws.
