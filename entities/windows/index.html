THREAT FILE · WINDOWS

Windows

Minimal Humanoid Reptile

Window entities are grey humanoid entities with bright red eyes and long claws. They seem to be very bony and skinny with reptile-like skin. This entity stays behind a window and will attempt to lure the player closer to them.

Survival card

DO

  • Stay away from the window until you know that there isn't a window entity there.
  • Walk into rooms in Level 188 instead of sprinting so you don't get caught off guard if you reach the window before the entity opens the curtains to show itself.

DON'T

  • Get too close to a window if the entity is there.
  • Run straight towards a window when entering any hotel room.
Sighted in
Level 188 Level 4

Appearance

Window entities are grey humanoid entities with bright red eyes and long claws. They seem to be very bony and skinny with reptile-like skin. This entity stays behind a window and will attempt to lure the player closer to them.

Location

This Entity can be found in:

  • Level 4
  • Level 188

Behavior

When passing by certain windows in Level 4 and Level 188, the Entity will pop up behind the window and stare. It will occasionally knock on the window to get someone's attention. If you get too close to it, it will break the window and kill you.

Do's & Don'ts

Do's

  • Stay away from the window until you know that there isn't a window entity there.
  • Walk into rooms in Level 188 instead of sprinting so you don't get caught off guard if you reach the window before the entity opens the curtains to show itself.

Don'ts

  • Get too close to a window if the entity is there.
  • Run straight towards a window when entering any hotel room.

Trivia

  • In Level 188, Window entities will only spawn behind windows that have views of the courtyard.
