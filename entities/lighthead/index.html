THREAT FILE · LIGHTHEAD

Lighthead

Moderate Humanoid with a Streetlight for a Head

Lighthead is a tall, skinny Entity that has a streetlight as a head. It is mostly brownish, and you would also be able to see it's light shining in the dark meaning you would know when it would be coming.

Sighted in
Level 2

Appearance

Lighthead is a tall, skinny Entity that has a streetlight as a head. It is mostly brownish, and you would also be able to see it's light shining in the dark meaning you would know when it would be coming.

Location

Lighthead was going to appear in:

  • Level 2

Behavior

Lighthead would have worked the same way as the Smiler in Level 2. Once you leave the darkness, it's light would be noticeable in the dark and then it would start chasing you. It looked to be way faster than The Smiler, so it would probably reach you a lot faster.

Trivia

  • Lighthead's removal was because Fancy was told that the Entity was not from the original Backrooms lore and Lighthead was replaced by the Smiler.
  • Lighthead only appeared in 2 Fancy Devlogs which included Episode 1, and in Episode 2 when it was removed.
  • The reason Lighthead was added was because Fancy thought it was cool to see it's Light shining in the dark.
