Appearance

The M.E.G. Base is a small abandoned M.E.G. Base located within The Hub. The Base has some desks with Computers, Storage Shelves, tables and drawers along with an Elevator that can be used to travel to three total places. There are windows with warnings, and on the other side is a view of Level 0. The Base is located at the end of a tunnel in The Hub.

Objective

Part 1: Enter the Base

Once you arrive to The Hub, you must enter The M.E.G. Base to unlock the Gate in Level 1. Before you can access it, you need to Solve a Puzzle. There is a Security Terminal next to the entrance, and it contains 3 slots full of a bunch of words. In order to complete it, match the words with the required code, and press the left mouse button to confirm. Do this 3 times and the door will open. When you first enter the Base, the lights will be out. You will need to find the control panel to turn on all the lights, which is on bottom right of the room. Simply flip all of the switches and the lights will turn on.

Part 2: Unlocking a Computer

In the Base, there are 4 Computers that you can access, but they require a password. A Note can be found on a table in the Level, and it contains 3 passwords. Two of them are crossed out, but the correct one isn't, which is itheardyou. Type this into the computer, and it will unlock.

Part 3: Opening the Gate

In the Computer, there is information on all the levels, but you need to have reached the Level in order to read the entry. There is an application in the computer that you can use to open up the gate in Level 1. It will first warn you, but if you proceed, it will say that the Gate is open, but nobody is authorized to perform rescue missions. And then you will be able to access the garage area of the Final Area. The information on all the levels on the Computers will also include your fastest escape after beating the respective Level, unless it has multiple areas.

The Major Explorers Group Base (The M.E.G. Base for short) is a base in Escape the Backrooms. It appears to be an Abandoned research outpost.

Items

There are no items that can be found in this Base.

Entities

There are no Entities in this Base.

Objects

Four objects can be found here. These objects include:

  • Security Terminal
  • Lights Panel
  • Computer
  • Storage Shelves

Entrances & Exits

Entrances

  • Enter the Base through the door on the left side of The Hub to get here.

Exits

  • Exit the Base the same way you came in to return to The Hub.
  • If you've discovered the Wrong Door Ending, the Elevator in the Base will be open and can lead be used to access the Level 3999 ending machine room, and the Bunker.

Trivia

  • This is currently the only Base in the Game.
  • You can see the windows in the Base in the background of the Story Mode and The Hub lobbies.
  • You can click anywhere to flip the switches on the Lights Panel.
  • You can skip the Computer password by having the game set to any language except for English.
  • In Fancy's 4th Devlog, he revealed that you will be able to store your Items in the Storage Shelves which is currently not possible so eventually, it might be possible in a future update.
